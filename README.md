# Temperature Converter

## Overview
This project is a Temperature Converter developed in Python as part of **Task 1** of the **Software Development Internship** at SkillCraft Technology.

The application allows users to convert temperatures between the three most commonly used temperature scales:

- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

## Features
- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- User-friendly command-line interface
- Accurate temperature conversion using standard formulas

## Technologies Used
- Python 3

## How It Works
The user selects the temperature scale to convert from and enters a temperature value. The program then calculates and displays the equivalent temperatures in the other two scales.

### Conversion Formulas

#### Celsius
- Fahrenheit = (C × 9/5) + 32
- Kelvin = C + 273.15

#### Fahrenheit
- Celsius = (F − 32) × 5/9
- Kelvin = Celsius + 273.15

#### Kelvin
- Celsius = K − 273.15
- Fahrenheit = (Celsius × 9/5) + 32
